# AndroidRobot API 概述
AndroidRobot API分对Native控件支持的API和对Webview控件支持的API。<br>

### Native API
### boolean click(int x, int y)
点击坐标<br>
参数: int x - x坐标 int y - y坐标<br>
返回值:true - 点击成功    false - 点击失败<br>

### click(String text)  
点击文本<br>
参数:String text - 文本文字<br>
返回值:true - 点击成功    false - 点击失败<br>
