# AndroidRobot
### 介绍
AndroidRobot是专门为移动无线端测试开发的一款自动化测试工具，它能够支持Android平台设备、APP等测试。</br>
目前支持以下功能：</br>
1，	清晰简洁的IDE界面风格</br>
2，	支持Python脚本</br>
3，	支持脚本编写、控件的识别、录制</br>
4，	支持Native、Webview控件识别</br>
5，	具有图像比较功能</br>
6，	具有任务管理功能、流畅的自动化引擎、清晰的输出报告</br>
